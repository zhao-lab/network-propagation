This folder contains data file and figures used to produce the results in the Network paper,
'different_penetration.xlsx' contains results on two sparse networks with different number of vehicles,(this one is used in the paper)
'restrict_degree' contains results on two networks with trimmed edges but same number of vehivles(50). (for detail, see 2017_8_22_meeting_ppt)
The 'eps' and 'png' files are used in the paper.
